## DRPS


### Setup
The project requires Java to run.


### Run
The application is compatible with Java 21 upwards.

#### Server
To test server using Java:
```
java -jar DRPS-0.0.1.jar -s
```

#### Client
To test server using Java:
```
java -jar DRPS-0.0.1.jar
```
